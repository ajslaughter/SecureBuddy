# Roadmap

Planned enhancements for WinSysAuto in future releases:

1. **Replica Domain Controller** – automate the build of a secondary DC for resiliency.
2. **WSUS Integration** – deploy Windows Server Update Services with baseline approval flows.
3. **Public Key Infrastructure** – bootstrap a two-tier AD CS deployment with templates.
4. **Microsoft Intune / Entra ID Hybrid** – integrate device management and cloud identities.
5. **Additional Health Tests** – extend reporting with replication, DFS, and certificate checks.
